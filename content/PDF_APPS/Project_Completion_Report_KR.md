# 📄 프로젝트 완료 보고서 (PDF 프로 편집기)

> 🇺🇸 [English Version](./Project_Completion_Report_EN)

### 1. 프로젝트 개요
본 프로젝트는 사용자가 민감한 정보가 포함된 PDF 문서를 안전하게 편집할 수 있도록 지원하는 **데스크톱 애플리케이션(PDF 프로 편집기)** 개발을 목표로 진행되었습니다. 단순한 시각적 가림 처리가 아닌 실제 데이터 구조에서 텍스트를 영구 삭제하는 '보안 마스킹' 기능과 사용자 정의 '텍스트 상자' 기능을 지원하여, 개인정보 보호 및 실무 효율성을 극대화합니다. 일일 작업 내역 및 업데이트에서는 특히 문제 해결(Troubleshooting)과 관련된 내용을 중심으로 기능 개선이 이루어졌습니다.

### 2. 아키텍처
* **Frontend (UI 환경):** `Python Tkinter` 및 `ttk`를 사용하여 Windows 환경에 이질감이 없는 네이티브 GUI를 구현했습니다. Canvas 위젯을 활용하여 PDF 렌더링, 줌(Zoom) 비율에 맞춘 화면 스크롤, 마우스 드래그를 통한 좌표 추적 등을 총괄합니다.
* **Backend (PDF 엔진):** 빠르고 안정적인 `PyMuPDF (fitz)` 라이브러리를 핵심 엔진으로 사용했습니다. PDF 렌더링(Pixmap 변환)부터 페이지 삽입/삭제, Redact(영구 삭제) 어노테이션 적용 등의 처리를 담당합니다.
* **이미지 처리:** PyMuPDF에서 추출된 Pixmap 데이터를 Tkinter에서 렌더링 가능한 이미지(ImageTk)로 변환하기 위해 `Pillow (PIL)` 라이브러리를 연계했습니다.

### 3. AI 활용 전략
* **지능형 코드 어시스턴트 적용:** Tkinter의 복잡한 Canvas 좌표계 설계 및 확대/축소(Zoom) 계산 로직을 구현하는 과정에서 생성형 AI를 활용한 신속한 프로토타이핑을 진행했습니다.
* **API 문제 해결(Troubleshooting):** PyMuPDF의 `add_redact_annot` 및 `apply_redactions()`를 통한 영구 삭제 로직 구현 시, AI와의 페어 프로그래밍을 통해 레퍼런스를 빠르게 분석하고 최적의 구현 방식을 도출했습니다.

### 4. 트러블슈팅 및 기술적 의사 결정
#### 일일 작업 및 주요 문제 해결 내역
* **문제 1: 마스킹된 텍스트의 유출 위험**
  * **상황:** 초기에는 단순히 검은색/빨간색 사각형을 글자 위에 덮어씌웠으나, 텍스트 드래그 및 복사가 가능하여 보안에 취약했습니다.
  * **해결:** PyMuPDF의 `add_redact_annot()`를 적용한 뒤, 명시적으로 `apply_redactions()` 함수를 호출하도록 로직을 변경했습니다. 이로써 시각적 가림뿐 아니라 원본 문서 데이터에서 해당 텍스트를 완전히(영구적으로) 파기하는 완벽한 보안 마스킹을 구현했습니다.
* **문제 2: 화면 확대/축소 시 캔버스 위치와 실제 PDF 좌표의 불일치**
  * **상황:** 사용자가 화면 배율(Zoom)을 변경하면 마우스로 드래그한 영역과 실제 PDF에 그려지는 도형의 크기가 맞지 않는 현상이 발생했습니다.
  * **해결:** 원본 PDF 페이지 크기와 렌더링된 픽셀(Pix) 크기를 실시간으로 비교하는 척도(`img_scale_x`, `img_scale_y`)를 도입하여, 캔버스 좌표를 실제 PDF 포인트 좌표로 정확히 변환하도록 구현했습니다.
* **문제 3: PDF 내 한글 폰트 깨짐 현상**
  * **상황:** 텍스트 상자 추가 시 영문은 정상 출력되나 한글 텍스트가 깨져서 PDF에 저장되는 문제가 있었습니다.
  * **해결:** Windows 기본 글꼴인 `malgun.ttf` (맑은 고딕)의 절대 경로를 탐색하여 `insert_textbox()`에 강제 적용(`fontname="malgun"`)함으로써 한글 인코딩 및 렌더링 문제를 해결했습니다.

### 5. 배포 및 향후 과제
* **배포 형태:** 현재는 단일 Python 스크립트(`PDF_APPS_1.py`)로 동작하며, 향후 배포 시 `PyInstaller` 등을 이용해 파이썬이 설치되지 않은 환경에서도 실행 가능한 독립형 실행 파일(.exe)로 패키징할 계획입니다.
* **향후 과제:**
  * **다중 작업 관리:** 현재 '실행 취소(Undo)'만 지원하므로, '다시 실행(Redo)' 기능을 추가.
  * **다중 탭(Multiple Tabs) 지원:** 여러 PDF 문서를 탭 형태로 동시에 띄워놓고 작업할 수 있는 환경 제공.
  * **자동 감지 및 마스킹 (AI 고도화):** 주민등록번호, 전화번호 등 정형화된 민감 정보를 식별하여 자동으로 마스킹 처리를 제안하는 기능 도입.
